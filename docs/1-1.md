## RED

### 목표 (단일 동작)

`Sample(sample_id, name, avg_production_time, yield_rate)`로 생성하면 각 필드가 그대로
보존되고, 초기 `stock`은 0이다. (PRD.md 2.1절)

### 작성할 실패 테스트

`tests/model/test_sample.py`

```python
from model.sample import Sample


def test_sample_생성_시_입력한_필드가_그대로_보존된다():
    sample = Sample(
        sample_id="SMP-001",
        name="Wafer-A",
        avg_production_time=2.5,
        yield_rate=0.9,
    )

    assert sample.sample_id == "SMP-001"
    assert sample.name == "Wafer-A"
    assert sample.avg_production_time == 2.5
    assert sample.yield_rate == 0.9


def test_sample_생성_시_초기_재고는_0이다():
    sample = Sample(
        sample_id="SMP-001",
        name="Wafer-A",
        avg_production_time=2.5,
        yield_rate=0.9,
    )

    assert sample.stock == 0
```

### 구현 방향 (GREEN 예고 — RED 단계에서는 작성하지 않음)

- `src/model/sample.py`에 `Sample`을 `dataclass`로 정의한다.
- 생성자 인자: `sample_id: str`, `name: str`, `avg_production_time: float`, `yield_rate: float`
- `stock: int = 0` — 등록 시 자동으로 0에서 시작 (PRD.md "초기 재고는 0" 규정).
- 이번 사이클에서는 `yield_rate` 범위 검증([1-2]) 등은 다루지 않는다 — 필드 보존/초기값만 검증한다.
