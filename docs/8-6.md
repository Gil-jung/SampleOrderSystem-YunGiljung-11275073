# [Phase 8] 사이클 8-6 — 출고 메뉴 흐름

`PLAN.md`의 Phase 8 [8-6] 항목에 대한 tdd 스킬 사이클 기록.

## RED

### 목표 (단일 동작)

`MenuController`의 출고 명령이 `ReleaseService`로 위임된다. (PRD.md 4.6절)

### 설계 메모 — 기존 테스트에 영향

`MenuController` 생성자에 `release_service`를 다섯 번째 필수 인자로 추가한다:
`MenuController(sample_service, order_service, monitoring_service, production_service, release_service)`.
기존 11개 테스트의 생성 호출부도 함께 조정한다.

### 작성할 실패 테스트

`tests/controller/test_menu_controller.py`에 아래 테스트를 추가한다.

```python
from service.release_service import ReleaseService


def test_출고_명령이_ReleaseService로_위임된다():
    sample_repository = SampleRepository()
    sample_repository.add(
        Sample(sample_id="SMP-001", name="Wafer-A", avg_production_time=2.5, yield_rate=0.9)
    )
    sample_repository.get("SMP-001").stock = 10
    order_repository = OrderRepository()
    production_service = ProductionService()
    sample_service = SampleService(sample_repository)
    order_service = OrderService(order_repository, sample_repository, production_service)
    monitoring_service = MonitoringService(order_repository, sample_repository)
    release_service = ReleaseService(order_repository)
    controller = MenuController(
        sample_service, order_service, monitoring_service, production_service, release_service
    )
    order_id = controller.reserve_order(
        sample_id="SMP-001", customer_name="홍길동", quantity=5
    )
    controller.approve_order(order_id)  # 재고 충분 -> CONFIRMED

    controller.release_order(order_id)

    assert order_repository.get(order_id).status == OrderStatus.RELEASE
```

기존 11개 테스트의 `MenuController(sample_service, order_service, monitoring_service, production_service)`도
`release_service`를 추가해 조정한다.

### 구현 방향 (GREEN 예고 — RED 단계에서는 작성하지 않음)

- `MenuController.__init__`에 `release_service` 인자 추가.
- `release_order(order_id)`: `self._release_service.release(order_id)` 위임.
