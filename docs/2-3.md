# [Phase 2] 사이클 2-3 — 전체 시료 조회

`PLAN.md`의 Phase 2 [2-3] 항목에 대한 tdd 스킬 사이클 기록.

## RED

### 목표 (단일 동작)

`SampleService.list_all()`이 등록된 모든 시료를 반환하며, 각 시료의 현재 재고
수량(`stock`)도 함께 확인할 수 있다. (PRD.md 4.1절 — "현재 재고 수량도 함께 표시")

### 작성할 실패 테스트

`tests/service/test_sample_service.py`에 아래 테스트를 추가한다.

```python
def test_전체_시료_조회_시_등록된_모든_시료와_재고가_반환된다():
    repository = SampleRepository()
    service = SampleService(repository)
    service.register(
        sample_id="SMP-001",
        name="Wafer-A",
        avg_production_time=2.5,
        yield_rate=0.9,
    )
    service.register(
        sample_id="SMP-002",
        name="Wafer-B",
        avg_production_time=1.0,
        yield_rate=0.8,
    )

    samples = service.list_all()

    ids = {sample.sample_id for sample in samples}
    assert ids == {"SMP-001", "SMP-002"}
    assert all(sample.stock == 0 for sample in samples)
```

### 구현 방향 (GREEN 예고 — RED 단계에서는 작성하지 않음)

- `SampleRepository.list()`: 저장된 모든 `Sample`을 리스트로 반환.
- `SampleService.list_all()`: `repository.list()`를 그대로 반환.

### 검증 방법

```bash
pytest tests/service/test_sample_service.py -v
```

→ `SampleRepository`/`SampleService`에 `list()`/`list_all()`이 아직 없으므로
`AttributeError`로 실패하는 것을 확인한다.
