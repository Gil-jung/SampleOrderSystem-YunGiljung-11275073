## RED

### 목표 (단일 동작)

`실 생산량 = ceil(부족분 / 수율)`를 계산하는 순수 함수. (PRD.md 4.5절)

### 작성할 실패 테스트

`tests/service/test_production_service.py`에 아래 테스트를 추가한다.

```python
import pytest

from service.production_service import calculate_actual_production


@pytest.mark.parametrize(
    "shortage, yield_rate, expected",
    [
        (9, 0.9, 10),   # 나누어떨어지는 경우: 9 / 0.9 = 10.0 -> ceil = 10
        (5, 0.9, 6),    # 나누어떨어지지 않는 경우: 5 / 0.9 = 5.555... -> ceil = 6
    ],
)
def test_실_생산량은_부족분을_수율로_나눈_값을_올림한다(shortage, yield_rate, expected):
    assert calculate_actual_production(shortage, yield_rate) == expected
```

### 구현 방향 (GREEN 예고 — RED 단계에서는 작성하지 않음)

- `src/service/production_service.py`에 모듈 수준 순수 함수
  `calculate_actual_production(shortage, yield_rate)`를 추가한다.
  `math.ceil(shortage / yield_rate)`를 반환한다.

### 검증 방법


→ `calculate_actual_production`이 아직 없으므로 `ImportError`로 실패하는 것을 확인한다.
