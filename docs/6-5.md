# [Phase 6] 사이클 6-5 — 시료별 재고 상태 판정 (고갈)

`PLAN.md`의 Phase 6 [6-5] 항목에 대한 tdd 스킬 사이클 기록.

## RED

### 목표 (단일 동작)

재고가 0이면 `MonitoringService.stock_status(sample_id)`가 `"고갈"`을 반환한다. (PRD.md 4.4절)

미출고 수요가 0인 상태에서 재고도 0인 경계 케이스(`0 >= 0`이라 현재 로직은 "여유"로
분류함)는 [6-6]에서 별도로 다룬다. 이번 사이클에서는 수요가 0보다 큰 상태에서 재고가
0인, 모호하지 않은 케이스로 검증한다.

### 작성할 실패 테스트

`tests/service/test_monitoring_service.py`에 아래 테스트를 추가한다.

```python
def test_재고가_0이면_고갈로_판정한다():
    order_repository = OrderRepository()
    sample_repository = SampleRepository()
    sample_repository.add(
        Sample(sample_id="SMP-001", name="Wafer-A", avg_production_time=2.5, yield_rate=0.9)
    )
    sample_repository.get("SMP-001").stock = 5
    order_service = OrderService(order_repository, sample_repository, ProductionService())
    monitoring_service = MonitoringService(order_repository, sample_repository)

    order_id = order_service.reserve(sample_id="SMP-001", customer_name="홍길동", quantity=5)
    order_service.approve(order_id)  # 재고 충분 -> CONFIRMED, stock 5 -> 0, 수요 5

    assert monitoring_service.stock_status("SMP-001") == "고갈"
```

### 구현 방향 (GREEN 예고 — RED 단계에서는 작성하지 않음)

- `stock_status`에 마지막 `else: return "고갈"` 분기를 추가한다. 앞선 두 분기(`여유`,
  `부족`)에서 걸러지지 않고 여기까지 오면 `stock == 0`이 보장된다 (수요 > 0인 경우에
  한해).

### 검증 방법

```bash
pytest tests/service/test_monitoring_service.py -v
```

→ 현재 `stock_status`는 이 케이스에서 `None`을 반환하므로, `assert None == "고갈"`로
실패하는 것을 확인한다.
