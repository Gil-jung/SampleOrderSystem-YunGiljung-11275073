# [Phase 10] 사이클 10-1 — 시료 관리 서브메뉴

`PLAN.md`의 Phase 10 [10-1] 항목에 대한 tdd 스킬 사이클 기록.

## RED

### 목표 (단일 동작 — 시료 관리 서브메뉴의 등록+조회 흐름)

메인 메뉴에서 `1`을 선택하면 시료 관리 서브메뉴(등록/조회/검색/뒤로가기)로 진입하고,
등록한 시료가 조회 목록에 표시된 뒤 `0`으로 메인 메뉴로 돌아가 다시 `0`으로 종료할 때까지
예외 없이 동작한다. `PoC/DataPersistence`의 `subprocess` 기반 스모크 테스트 패턴을
그대로 사용한다.

### 작성할 실패 테스트

`tests/test_main.py`에 아래 테스트를 추가한다.

```python
def test_시료_관리_메뉴에서_등록한_시료가_목록_조회에_표시된다():
    user_input = "\n".join(
        [
            "1",  # 메인 메뉴 -> 시료 관리
            "1",  # 시료 등록
            "SMP-001",
            "Wafer-A",
            "2.5",
            "0.9",
            "2",  # 시료 목록 조회
            "0",  # 시료 관리 메뉴에서 뒤로가기
            "0",  # 메인 메뉴 종료
            "",
        ]
    )

    result = subprocess.run(
        [sys.executable, "main.py"],
        input=user_input,
        capture_output=True,
        text=True,
        cwd=SRC_DIR,
    )

    assert result.returncode == 0
    assert result.stderr == ""
    assert "Wafer-A" in result.stdout
    assert "SMP-001" in result.stdout
```

### 구현 방향 (GREEN 예고 — RED 단계에서는 작성하지 않음)

- `ConsoleView`에 추가:
  - `render_sample_menu()`: "1. 등록 / 2. 조회 / 3. 검색 / 0. 뒤로가기" 형태의 서브메뉴 문자열.
  - `render_sample_list(samples)`: `list_samples()`/`search_samples()`가 반환한 `dict`
    리스트를 사람이 읽을 문자열로 렌더링 (시료 없음일 때도 처리).
- `main.py`에 `handle_sample_menu(controller, view)` 함수를 추가하고, 메인 루프에서
  `choice == "1"`이면 이 함수를 호출하도록 분기를 추가한다.
  - 서브메뉴에서 `1`(등록)은 `input()`으로 `sample_id`/`name`/`avg_production_time`/
    `yield_rate`를 순서대로 받아 `controller.register_sample(...)`을 호출한다.
  - `2`(조회)는 `controller.list_samples()` 결과를 `view.render_sample_list(...)`로 출력한다.
  - `3`(검색)은 이번 사이클 테스트가 요구하지 않으므로 최소 뼈대만 두거나, `2`와
    유사하게 바로 구현해도 무방하다 (과도한 설계 금지 원칙에 따라, 테스트가 검증하는
    등록+조회 흐름만 최소로 구현하고 검색은 동일 패턴으로 함께 넣되 별도 테스트는 이후
    필요시 추가).
  - `0`은 서브메뉴 루프를 빠져나와 메인 메뉴로 복귀한다.

### 검증 방법

```bash
pytest tests/test_main.py -v
```

→ `main.py`가 아직 `choice == "1"` 분기를 처리하지 않으므로(현재는 `"0"` 외 입력을
무시하고 메인 메뉴를 다시 표시), 서브메뉴 프롬프트("등록"/"조회" 등)가 표시되지 않아
`assert "Wafer-A" in result.stdout`가 실패하는 것을 확인한다.
