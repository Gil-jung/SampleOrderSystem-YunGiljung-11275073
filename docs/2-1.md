# [Phase 2] 사이클 2-1 — 시료 등록

`PLAN.md`의 Phase 2 [2-1] 항목에 대한 tdd 스킬 사이클 기록.

## RED

### 목표 (단일 동작)

`SampleService.register(sample_id, name, avg_production_time, yield_rate)`를 호출하면
`SampleRepository`에 저장되고, 저장된 시료의 초기 재고는 0이다. (PRD.md 4.1절)

### 설계 메모

- `PLAN.md`는 `Repository`와 `Service`를 별도 계층으로 두지만, 사이클 목록에는 Repository
  전용 사이클이 없다. Repository는 이번 사이클에서 Service가 실제로 사용하는 협력 객체로
  함께 도입하고(mock 아님 — 실제 객체), 별도 단위 테스트는 만들지 않는다. Repository의
  `add`/`get` 계약은 이번 테스트가 실제로 `SampleRepository` 인스턴스를 통해 저장/조회하는
  것으로 충분히 검증된다.
- `SampleService`는 생성자로 `SampleRepository` 인스턴스를 주입받는다(의존성 주입).
- `SampleService.get(sample_id)`처럼 별도 조회 API를 이번 사이클에서 추가하지 않는다
  ([2-3] 전체 시료 조회에서 다룬다). 대신 등록 결과 검증은 `SampleRepository.get()`을
  직접 호출해 확인한다 — Service의 내부 구현이 아니라 "실제로 저장소에 반영되었는가"라는
  관찰 가능한 결과를 검증하는 것이므로 안티패턴이 아니다.

### 작성할 실패 테스트

`tests/service/test_sample_service.py` (신규 파일)

```python
from repository.sample_repository import SampleRepository
from service.sample_service import SampleService


def test_시료_등록_시_초기_재고_0으로_저장된다():
    repository = SampleRepository()
    service = SampleService(repository)

    service.register(
        sample_id="SMP-001",
        name="Wafer-A",
        avg_production_time=2.5,
        yield_rate=0.9,
    )

    stored = repository.get("SMP-001")
    assert stored.sample_id == "SMP-001"
    assert stored.name == "Wafer-A"
    assert stored.avg_production_time == 2.5
    assert stored.yield_rate == 0.9
    assert stored.stock == 0
```

### 구현 방향 (GREEN 예고 — RED 단계에서는 작성하지 않음)

- `src/repository/sample_repository.py`: `SampleRepository`
  - `add(sample: Sample) -> None`: `dict[sample_id, Sample]`에 저장
  - `get(sample_id: str) -> Sample`: 저장된 시료 반환 (없을 때 처리는 이번 사이클 범위 밖)
- `src/service/sample_service.py`: `SampleService`
  - `__init__(self, repository: SampleRepository)`
  - `register(sample_id, name, avg_production_time, yield_rate) -> None`: `Sample`을 생성해
    `repository.add()`로 저장 (초기 재고 0은 [1-1]에서 이미 `Sample`의 기본값으로 보장됨)

## GREEN

`src/repository/sample_repository.py`와 `src/service/sample_service.py`를 최소 구현했다.

```python
# repository/sample_repository.py
class SampleRepository:
    def __init__(self):
        self._samples = {}

    def add(self, sample):
        self._samples[sample.sample_id] = sample

    def get(self, sample_id):
        return self._samples[sample_id]
```

```python
# service/sample_service.py
from model.sample import Sample


class SampleService:
    def __init__(self, repository):
        self._repository = repository

    def register(self, sample_id, name, avg_production_time, yield_rate):
        sample = Sample(
            sample_id=sample_id,
            name=name,
            avg_production_time=avg_production_time,
            yield_rate=yield_rate,
        )
        self._repository.add(sample)
```
