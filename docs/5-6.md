# [Phase 5] 사이클 5-6 — 생산 진행 상황 조회

`PLAN.md`의 Phase 5 [5-6] 항목에 대한 tdd 스킬 사이클 기록.

## RED

### 목표 (단일 동작)

`ProductionService.current_status()`가 현재 생산 중인(큐 맨 앞) 주문의 생산 정보
(주문 ID, 실 생산량, 총 생산 시간)를 반환한다. (PRD.md 4.5절 — "표기할 정보 수준은
자율적으로 결정")

### 설계 메모 — 기존 계약을 깨지 않는 확장

- 지금까지 `enqueue(order_id)`는 `order_id`만 큐에 쌓았다. 생산 현황을 표기하려면
  실 생산량/총 생산 시간이 필요한데, 이 값은 [5-1]~[5-3]에서 이미 만든 순수 함수
  (`calculate_shortage`, `calculate_actual_production`, `calculate_total_production_time`)로
  계산 가능하다. `OrderService.approve`가 이 함수들을 호출해 계산한 값을 `enqueue`에
  **선택적(optional) 키워드 인자**로 함께 전달하도록 확장한다:
  `enqueue(order_id, actual_production=None, total_production_time=None)`.
  기본값을 `None`으로 두어 기존 `enqueue(order_id)` 호출부(과거 사이클의 코드는 없고,
  현재는 `OrderService.approve` 내부 호출 한 곳뿐)에 영향이 없도록 한다.
- `list_queue()`는 지금처럼 `order_id` 리스트를 그대로 반환한다(FIFO 순서 검증용,
  [5-4]/[5-5]에서 이미 검증됨) — 반환 타입을 바꾸지 않는다. 생산 정보는 별도의
  `current_status()`로만 노출한다.
- `current_status()`는 큐가 비어있으면 `None`을, 그렇지 않으면 맨 앞 주문의 정보를
  `dict`(`order_id`, `actual_production`, `total_production_time`)로 반환한다.

    service = OrderService(order_repository, sample_repository, production_service)
    order_id = service.reserve(sample_id="SMP-001", customer_name="홍길동", quantity=9)

### 구현 방향 (GREEN 예고 — RED 단계에서는 작성하지 않음)

- `ProductionService`:
  - `__init__`에 `self._status = {}` 추가.
  - `enqueue(order_id, actual_production=None, total_production_time=None)`: 큐에 추가하고
    `self._status[order_id]`에 `{"order_id": ..., "actual_production": ..., "total_production_time": ...}` 저장.
  - `current_status()`: 큐가 비어있으면 `None`, 아니면 `self._status[self._queue[0]]` 반환.
- `OrderService.approve`의 재고 부족 분기에서 `calculate_shortage` → `calculate_actual_production`
  → `calculate_total_production_time`을 순서대로 호출해 `enqueue`에 전달한다.

## GREEN

`ProductionService`에 `_status`와 `current_status()`를 추가하고, `enqueue`에 선택적
키워드 인자를 추가했다. `OrderService.approve`가 [5-1]~[5-3]의 순수 함수를 실제로
조합해 사용하도록 했다.

```python
# service/production_service.py
class ProductionService:
    def __init__(self):
        self._queue = []
        self._status = {}

    def enqueue(self, order_id, actual_production=None, total_production_time=None):
        self._queue.append(order_id)
        self._status[order_id] = {
            "order_id": order_id,
            "actual_production": actual_production,
            "total_production_time": total_production_time,
        }

    def current_status(self):
        if not self._queue:
            return None
        return self._status[self._queue[0]]
```

```python
# service/order_service.py (approve 내부, 재고 부족 분기)
shortage = calculate_shortage(order.quantity, sample.stock)
actual_production = calculate_actual_production(shortage, sample.yield_rate)
total_production_time = calculate_total_production_time(
    sample.avg_production_time, actual_production
)
order.transition_to(OrderStatus.PRODUCING)
self._production_service.enqueue(order_id, actual_production, total_production_time)
```
