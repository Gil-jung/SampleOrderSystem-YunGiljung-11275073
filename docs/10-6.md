# [Phase 10] 사이클 10-6 — 메인 메뉴에 종료 항목 표시

`PLAN.md`의 Phase 10 [10-6] 항목에 대한 tdd 스킬 사이클 기록.

## RED

### 목표 (단일 동작)

`ConsoleView.render_main_menu`가 렌더링하는 메인 메뉴에 `"0. 종료"` 항목이 포함된다.
`main.py`는 이미 `"0"` 입력 시 종료하도록 구현되어 있지만([8-7]), 메뉴 화면에는
이 선택지가 안내되지 않고 있었다.

### 작성할 실패 테스트

`tests/view/test_console_view.py`에 아래 테스트를 추가한다.

```python
def test_메인_메뉴에_종료_항목이_표시된다():
    view = ConsoleView()

    output = view.render_main_menu({"sample_count": 0, "total_stock": 0})

    assert "0. 종료" in output
```

### 구현 방향 (GREEN 예고 — RED 단계에서는 작성하지 않음)

- `ConsoleView.render_main_menu`의 메뉴 항목 리스트 마지막에 `"0. 종료"`를 추가한다.

### 검증 방법

```bash
pytest tests/view/test_console_view.py -v
```

→ 현재 메뉴에는 `"0. 종료"` 항목이 없으므로 `assert "0. 종료" in output`이 실패하는
것을 확인한다.
