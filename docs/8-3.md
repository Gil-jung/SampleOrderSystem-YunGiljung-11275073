## RED

### 목표 (단일 동작 — 예약/목록조회/승인/거절 4개 명령의 위임)

`MenuController`의 주문 관련 명령(예약, 접수된 주문 목록 조회, 승인, 거절)이
`OrderService`로 올바르게 위임되고, 목록 조회는 View용 `dict` 리스트로 반환된다.
(PRD.md 3절 "주문 (접수 / 승인 / 거절)" 메뉴)

### 설계 메모 — 기존 테스트에 영향

`MenuController`가 주문 관련 명령을 처리하려면 `OrderService`를 알아야 하므로,
생성자에 두 번째 필수 인자로 추가한다: `MenuController(sample_service, order_service)`.
[8-2]에서 작성한 기존 3개 테스트의 `MenuController(sample_service)` 생성 호출부도
`order_service`를 함께 전달하도록 조정한다 ([3-2]/[4-3]/[6-3]과 동일한 패턴).

### 작성할 실패 테스트

`tests/controller/test_menu_controller.py`에 아래 테스트를 추가한다.

```python
from model.order import OrderStatus
from model.sample import Sample
from repository.order_repository import OrderRepository
from service.order_service import OrderService
from service.production_service import ProductionService


def test_주문_예약_명령이_OrderService로_위임된다():
    sample_repository = SampleRepository()
    sample_repository.add(
        Sample(sample_id="SMP-001", name="Wafer-A", avg_production_time=2.5, yield_rate=0.9)
    )
    order_repository = OrderRepository()
    sample_service = SampleService(sample_repository)
    order_service = OrderService(order_repository, sample_repository, ProductionService())
    controller = MenuController(sample_service, order_service)

    order_id = controller.reserve_order(
        sample_id="SMP-001", customer_name="홍길동", quantity=5
    )

    stored = order_repository.get(order_id)
    assert stored.customer_name == "홍길동"
    assert stored.status == OrderStatus.RESERVED


def test_접수된_주문_목록_조회_명령이_dict_리스트로_반환된다():
    sample_repository = SampleRepository()
    sample_repository.add(
        Sample(sample_id="SMP-001", name="Wafer-A", avg_production_time=2.5, yield_rate=0.9)
    )
    order_repository = OrderRepository()
    sample_service = SampleService(sample_repository)
    order_service = OrderService(order_repository, sample_repository, ProductionService())
    controller = MenuController(sample_service, order_service)
    order_id = controller.reserve_order(
        sample_id="SMP-001", customer_name="홍길동", quantity=5
    )

    result = controller.list_reserved_orders()

    assert result == [
        {
            "order_id": order_id,
            "sample_id": "SMP-001",
            "customer_name": "홍길동",
            "quantity": 5,
            "status": "RESERVED",
        }
    ]


def test_주문_승인_명령이_OrderService로_위임된다():
    sample_repository = SampleRepository()
    sample_repository.add(
        Sample(sample_id="SMP-001", name="Wafer-A", avg_production_time=2.5, yield_rate=0.9)
    )
    sample_repository.get("SMP-001").stock = 10
    order_repository = OrderRepository()
    sample_service = SampleService(sample_repository)
    order_service = OrderService(order_repository, sample_repository, ProductionService())
    controller = MenuController(sample_service, order_service)
    order_id = controller.reserve_order(
        sample_id="SMP-001", customer_name="홍길동", quantity=5
    )

    controller.approve_order(order_id)

    assert order_repository.get(order_id).status == OrderStatus.CONFIRMED


def test_주문_거절_명령이_OrderService로_위임된다():
    sample_repository = SampleRepository()
    sample_repository.add(
        Sample(sample_id="SMP-001", name="Wafer-A", avg_production_time=2.5, yield_rate=0.9)
    )
    order_repository = OrderRepository()
    sample_service = SampleService(sample_repository)
    order_service = OrderService(order_repository, sample_repository, ProductionService())
    controller = MenuController(sample_service, order_service)
    order_id = controller.reserve_order(
        sample_id="SMP-001", customer_name="홍길동", quantity=5
    )

    controller.reject_order(order_id)

    assert order_repository.get(order_id).status == OrderStatus.REJECTED
```

기존 [8-2] 테스트 3개의 `MenuController(sample_service)`도
`MenuController(sample_service, order_service)`로 조정한다 (더미 `OrderService` 인스턴스 전달).

### 구현 방향 (GREEN 예고 — RED 단계에서는 작성하지 않음)

- `MenuController.__init__(self, sample_service, order_service)`로 변경.
- `reserve_order(sample_id, customer_name, quantity)`: `order_service.reserve(...)` 그대로 위임,
  반환된 `order_id`를 그대로 반환.
- `list_reserved_orders()`: `order_id`가 필요한데 `Order` 모델에는 `order_id` 필드가
  없으므로([1-3]의 YAGNI 결정), `OrderRepository`에 `(order_id, order)` 쌍을 함께
  반환하는 `list_with_ids()`를 추가하고, `OrderService.list_reserved_with_ids()`가
  이를 `RESERVED` 상태로 필터링해 반환하도록 한다. `MenuController.list_reserved_orders()`는
  그 결과를 `[{"order_id": ..., "sample_id": ..., "customer_name": ..., "quantity": ..., "status": status.name}]`로 변환한다.
- `approve_order(order_id)`: `order_service.approve(order_id)` 위임.
- `reject_order(order_id)`: `order_service.reject(order_id)` 위임.

## GREEN

`OrderRepository.list_with_ids()`, `OrderService.list_reserved_with_ids()`를 추가하고,
`MenuController`에 `order_service`와 4개 메서드(예약/목록조회/승인/거절)를 추가했다.

```python
# repository/order_repository.py
def list_with_ids(self):
    return list(self._orders.items())
```

```python
# service/order_service.py
def list_reserved_with_ids(self):
    return [
        (order_id, order)
        for order_id, order in self._order_repository.list_with_ids()
        if order.status == OrderStatus.RESERVED
    ]
```

```python
# controller/menu_controller.py
def reserve_order(self, sample_id, customer_name, quantity):
    return self._order_service.reserve(sample_id, customer_name, quantity)

def list_reserved_orders(self):
    return [
        self._order_to_dict(order_id, order)
        for order_id, order in self._order_service.list_reserved_with_ids()
    ]

def approve_order(self, order_id):
    self._order_service.approve(order_id)

def reject_order(self, order_id):
    self._order_service.reject(order_id)
```
