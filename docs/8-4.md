# [Phase 8] 사이클 8-4 — 모니터링 메뉴 흐름

`PLAN.md`의 Phase 8 [8-4] 항목에 대한 tdd 스킬 사이클 기록.

## RED

### 목표 (단일 동작 — 주문량/재고량 조회 2개 명령의 위임)

`MenuController`의 모니터링 명령(주문량 조회, 재고량 조회)이 `MonitoringService`로
위임되고, View용 순수 자료구조(`dict`)로 변환되어 반환된다. (PRD.md 4.4절)

### 설계 메모 — 기존 테스트에 영향

`MenuController` 생성자에 `monitoring_service`를 세 번째 필수 인자로 추가한다:
`MenuController(sample_service, order_service, monitoring_service)`. [8-2]/[8-3]에서
작성한 기존 7개 테스트의 생성 호출부도 `monitoring_service`를 함께 전달하도록 조정한다.

### 작성할 실패 테스트

`tests/controller/test_menu_controller.py`에 아래 테스트를 추가한다.

```python
from service.monitoring_service import MonitoringService


def test_주문량_조회_명령이_상태명_키의_dict로_반환된다():
    sample_repository = SampleRepository()
    sample_repository.add(
        Sample(sample_id="SMP-001", name="Wafer-A", avg_production_time=2.5, yield_rate=0.9)
    )
    order_repository = OrderRepository()
    sample_service = SampleService(sample_repository)
    order_service = OrderService(order_repository, sample_repository, ProductionService())
    monitoring_service = MonitoringService(order_repository, sample_repository)
    controller = MenuController(sample_service, order_service, monitoring_service)
    controller.reserve_order(sample_id="SMP-001", customer_name="홍길동", quantity=1)

    result = controller.get_order_counts()

    assert result["RESERVED"] == 1
    assert result["CONFIRMED"] == 0
    assert result["PRODUCING"] == 0
    assert result["RELEASE"] == 0


def test_재고량_조회_명령이_시료별_재고_상태_리스트로_반환된다():
    sample_repository = SampleRepository()
    sample_repository.add(
        Sample(sample_id="SMP-001", name="Wafer-A", avg_production_time=2.5, yield_rate=0.9)
    )
    order_repository = OrderRepository()
    sample_service = SampleService(sample_repository)
    order_service = OrderService(order_repository, sample_repository, ProductionService())
    monitoring_service = MonitoringService(order_repository, sample_repository)
    controller = MenuController(sample_service, order_service, monitoring_service)

    result = controller.get_stock_statuses()

    assert result == [{"sample_id": "SMP-001", "stock": 0, "status": "여유"}]
```

기존 [8-2]/[8-3] 테스트 7개의 `MenuController(sample_service, order_service)`도
`MenuController(sample_service, order_service, monitoring_service)`로 조정한다.

### 구현 방향 (GREEN 예고 — RED 단계에서는 작성하지 않음)

- `MenuController.__init__(self, sample_service, order_service, monitoring_service)`로 변경.
- `get_order_counts()`: `monitoring_service.count_by_status()` 결과(키가 `OrderStatus`
  Enum)를 `{status.name: count for status, count in ...}`로 변환.
- `get_stock_statuses()`: `sample_service.list_all()`을 순회하며 각 시료에 대해
  `{"sample_id": ..., "stock": ..., "status": monitoring_service.stock_status(sample_id)}`를
  구성.
